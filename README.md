# npm-toggle
:hammer: npm源切换工具

### 安装 (linux下非root用户需要sudo)
```
[sudo] npm i npm-toggle -g 
```

### 使用
**切换淘宝源**
```
toggle -t
```
**切换官方源**
```
toggle -n
```
**查看当前源**
```
toggle -r
```
**自动切换源并发布**
```
toggle -p
```
